## Features
- Encrypted user authorizaton
- Database initialization
- Login/logout
- User settings
- Modern UI
- No CSS/JS dependencies
- Easily customizable

## Setup
``` 
git clone https://github.com/anfederico/Flaskex
cd Flaskex
pip install -r requirements.txt
python app.py
```
